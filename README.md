# Portfolio

A modern portfolio website built with React, TypeScript, and Tailwind CSS. This is my personal portfolio showcasing my backend development skills, DevOps projects, and experience in building scalable applications.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Supabase

## Features

- Responsive design
- Dark mode with orange accent
- Contact form with Supabase integration
- Smooth animations
- Mobile-friendly
